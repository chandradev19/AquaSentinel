# AquaShield AI - Admin Manual

## Overview
The Admin role is designed for health ministry officials, district medical officers, and system administrators. Admins have global visibility over the entire National Health Intelligence System.

## Login
- **URL:** `/login`
- **Default Credentials:** `admin@aquashield.gov` / `admin123`

## Key Features

### 1. National Command Center (War Room)
- **Navigation:** Sidebar -> `Dashboard` or `AI Command Center`
- **Function:** Provides a macroscopic view of the nation's health.
- **Components:**
  - **National Health Index:** An aggregate score of overall safety.
  - **Live India Risk Map:** Geospatial mapping of all active monitoring nodes.
  - **Disease Trends:** Area charts forecasting the rise and fall of infections.

### 2. Village Digital Twin
- **Navigation:** Sidebar -> `Villages` -> Click on a Village Name
- **Function:** A localized dashboard for a specific geography.
- **Components:**
  - View the assigned field workers.
  - Monitor the real-time timeline of Citizen Reports and Water Tests.
  - Export the **Village Report PDF**.

### 3. Outbreak Simulator
- **Navigation:** Sidebar -> `AI Command Center` -> `Outbreak Simulator`
- **Function:** Run "What-If" scenarios using the predictive AI.
- **Usage:** Adjust the Rainfall and Contamination sliders, then click `Run AI Simulation` to see how epidemic risks will change.

### 4. Alert Management
- **Navigation:** Sidebar -> `Alerts`
- **Function:** View all automated emergency broadcasts. Admins can manually resolve alerts once field workers confirm mitigation.

### 5. SIH Presentation Mode
- **Navigation:** Sidebar (Bottom Left) -> `START SIH DEMO`
- **Function:** Triggers a 5-minute automated, hands-free guided tour of the entire application for hackathon judges.
